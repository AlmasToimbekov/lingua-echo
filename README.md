# LinguaEcho

Простое приложение для изучения английского языка всей семьёй (в первую очередь — для детей).  
Дети могут слушать, выделять кусочки фразы, зацикливать их на медленной скорости и генерировать новые полезные шаблоны с живыми голосами.

## Текущее состояние (актуально на момент последнего обновления)

- **По умолчанию** при первом запуске и после сброса — **2 шаблона** (не 10, чтобы не перегружать).
- **Вертикальный список шаблонов** слева (удобно скроллить).
- **Большой английский плеер**:
  - Waveform (wavesurfer.js).
  - Точная прокрутка + кнопки быстрого перемотки назад (|<<, -1s, -3s, -5s).
  - Выбор скорости (кнопки увеличены для детских пальцев).
  - **Удобная кнопка выделения региона** для детей: «Выделить первые 0,5с» (создаёт регион с начала фразы, виден сразу; потом легко растянуть/сдвинуть drag'ом по waveform).
  - Зацикливание выделенного фрагмента.
  - На узких экранах (iPhone портрет) список шаблонов/папок прячется по умолчанию — фокус на фразе + крупные кнопки «предыдущий/следующий» для линейного прохода.
- **Простая русская озвучка** — одна большая кнопка справа.
- **Генерация новых шаблонов** через Google Gemini (требуется ключ пользователя).
- **Озвучка** через ElevenLabs (рекомендуется, требуется ключ) или браузерный синтез (fallback без ключа).
- **Персистентность аудио**: тексты + реальные байты голосов (EN и RU) сохраняются между обновлениями страницы и перезапусками браузера благодаря localStorage + IndexedDB. После reload голоса не исчезают.
- Полностью работает без ключей (демо-режим на браузерных голосах).
- Никаких аккаунтов и сервера в текущей версии — всё хранится локально у пользователя.

## Быстрый старт

```bash
npm install
npm run dev
```

Откройте http://localhost:3000

### Что попробовать
1. Переключайте шаблоны в левом списке.
2. Выделяйте кусочки английской фразы кнопкой «Выделить первые 0,5с» (или перетаскиванием по waveform для создания/правки региона) → зацикливайте на 0.75×.
3. Нажмите шестерёнку → вставьте ключ Google Gemini (для генерации текстов) и/или ElevenLabs (для качественных голосов).
4. Нажмите «Сгенерировать шаблоны» — укажите тему, сложность и количество (от 1).

## Ключи (Bring Your Own Key)

- **Google Gemini** — для генерации текстов шаблонов (бесплатный tier обычно хватает).  
  Получить: https://aistudio.google.com/app/apikey
- **ElevenLabs** — для естественных американских + русских голосов (рекомендуется).  
  Получить: https://elevenlabs.io/app/developers/api-keys
- Ключи хранятся только в браузере пользователя (localStorage). Приложение их никуда не отправляет, кроме прямых запросов к API от имени пользователя.

Без ключей приложение тоже работает (браузерные голоса + предустановленные шаблоны).

## Как устроена персистентность аудио (важно!)

При генерации/перегенерации мы получаем Blob от TTS и:
- Создаём временный object URL для текущей сессии (для wavesurfer и <audio>).
- Сохраняем сами байты (ArrayBuffer) в IndexedDB.

При загрузке страницы:
- Тексты берём из localStorage.
- Для каждого шаблона загружаем буферы из IndexedDB и создаём **новые** свежие object URL.

Благодаря этому после F5 / закрытия браузера сгенерированные голоса остаются на месте. Старые blob: URL никогда не сохраняются надолго.

См. `lib/audioStorage.ts` + гидрацию в `app/page.tsx`.

## Для детей сделано удобно

- Крупный текст.
- Большие тап-зоны (особенно скорость и кнопки выделения региона).
- Простая кнопка «Выделить первые 0,5с» — создаёт видимый регион сразу (даже если аудио не играло), не нужно уметь точно драгить сначала.
- Папки («Изученные» по умолчанию + пользовательские) + кнопка «Отметить как изученное» и «Переместить в папку…» (с drag & drop на десктопе) для организации растущей коллекции.
- На телефонах список шаблонов можно прятать; для быстрого переключения — большие кнопки «← →» прямо над текстом фразы.
- Возможность зацикливать короткий кусочек на 0.6–0.75×.
- Минимум 1 шаблон в генерации (можно даже по одному добавлять).

## Публикация в веб (Vercel)

Приложение полностью клиентское — его очень легко выложить в интернет, чтобы дети могли открыть по ссылке с любого устройства.

**Рекомендуемый способ (первый раз):**
1. Создай аккаунт на [Vercel](https://vercel.com) (можно войти через GitHub).
2. Запушь этот репозиторий на GitHub.
3. На Vercel нажми **Add New Project** → импортируй свой GitHub-репозиторий.
4. Vercel автоматически увидит, что это Next.js. Никакие переменные окружения настраивать не нужно.
5. Нажми Deploy.

После деплоя получишь публичную ссылку вида `https://lingua-echo-xxx.vercel.app`.

**Важно понимать:**
- Все данные (шаблоны, сгенерированные голоса) хранятся только в браузере конкретного устройства (localStorage + IndexedDB). Нет синхронизации между телефонами/планшетами.
- Ключи Gemini и ElevenLabs пользователь вводит сам в разделе «Настройки» — они никуда не уходят с устройства.
- Это идеально подходит для текущей версии. Для семейной синхронизации нужен будет бэкенд (Phase 2).

При каждом `git push` на main Vercel будет автоматически обновлять приложение.

Подробную инструкцию с нуля смотри в чате с ИИ (или попроси обновить этот раздел).
